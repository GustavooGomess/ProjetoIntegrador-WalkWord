# 🚀 WalkWord Smart Store

### Projeto Integrador 2025/02 - SENAI

---

## 📌 Sobre o Projeto

O **WalkWord Smart Store** é uma aplicação web desenvolvida como parte do **Projeto Integrador**, com o objetivo de criar uma solução inovadora para a loja WalkWord, especializada na venda de roupas **novas e seminovas**, com foco em **moda sustentável**.

A plataforma integra tecnologias modernas como **Inteligência Artificial, Big Data, Frontend, Backend, Engenharia de Software e Redes**, proporcionando uma experiência de compra **personalizada, segura e eficiente**.

---

## 🎯 Objetivo

### Objetivo Geral

Desenvolver uma plataforma web inteligente para e-commerce de moda sustentável, oferecendo uma experiência moderna e personalizada ao usuário.

### Objetivos Específicos

* Criar uma interface responsiva e intuitiva
* Implementar sistema de recomendação com IA
* Garantir segurança e escalabilidade da aplicação
* Otimizar performance com boas práticas de redes
* Incentivar o consumo sustentável

---

## 🏭 Problema

A loja WalkWord necessita de uma solução digital moderna que:

* Melhore a experiência de compra online
* Ofereça recomendações personalizadas
* Garanta segurança nas transações
* Otimize o desempenho da aplicação
* Incentive a compra de roupas seminovas

---

## 💡 Solução Proposta

Desenvolvimento de uma aplicação web inteligente que:

* Oferece navegação rápida e intuitiva
* Recomenda produtos com base no comportamento do usuário
* Permite compra segura via múltiplos meios de pagamento
* Promove a sustentabilidade através da economia circular

---

## ⚙️ Funcionalidades

* 👤 Cadastro e login de usuários
* 🛍️ Catálogo de produtos (novos e seminovos)
* 🔍 Filtros avançados (tamanho, cor, estado da peça)
* 📄 Página de produto com imagens em alta resolução e descrição detalhada
* 🤖 Recomendações personalizadas com IA
* 💳 Integração com pagamentos (Pix, cartão, boleto)
* 📦 Carrinho de compras e gerenciamento de pedidos

---
